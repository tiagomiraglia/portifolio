# Portfólio Tiago Miraglia

Portfólio profissional desenvolvido em HTML e CSS, apresentando trajetória, experiência e habilidades em Tecnologia da Informação.

## Sobre o Projeto

Este portfólio foi criado para demonstrar conhecimentos em desenvolvimento front-end e apresentar de forma elegante e profissional a jornada de mais de 20 anos na área de TI.

## Funcionalidades

- Design responsivo e moderno
- Navegação intuitiva
- Timeline interativa da trajetória profissional
- Seções de experiência, habilidades e projetos
- Links para projetos no GitHub
- Layout otimizado para diferentes dispositivos

## Tecnologias Utilizadas

- HTML5
- CSS3
- Design Responsivo
- Gradientes e Animações CSS

## Como Visualizar

1. Clone o repositório ou baixe os arquivos
2. Abra o arquivo `index.html` no navegador
3. Navegue pelas seções usando o menu superior

## Estrutura do Projeto

```
portfolio/
├── index.html    # Página principal
├── style.css     # Estilos e layout
└── README.md     # Documentação
```

## Seções

- **Hero**: Apresentação inicial com call-to-action
- **Sobre**: Timeline da trajetória profissional
- **Experiência**: Detalhamento das experiências profissionais
- **Habilidades**: Stack técnico e competências
- **Projetos**: Portfólio de projetos desenvolvidos
- **Contato**: Informações de contato e redes
